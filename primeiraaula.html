Adobe DreamWeaver

O que é e o que podemos fazer com DreamWeaver?  

O Adobe Dreamweaver é um programa de design web que permite criar, programar e gerenciar sites. Ele é uma ferramenta popular para web designers e desenvolvedores de front-end.

Com o Adobe Dreamweaver é possível criar, editar, gerenciar e desenvolver sites, páginas e aplicativos para a web.

Como trabalhamos ou fazemos.

Temos duas opções de usarmos o Dreamweaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (comandos), ou criarmos o site programando (TAGs) com códigos em HTML, CSS e outras linguagens de programação Web.
